Task Management Application Documentation

Purpose

The Task Management Application is designed to help users manage their tasks efficiently. It provides basic CRUD (Create, Read, Update, Delete) functionalities, allowing users to create, edit, and filter tasks based on their status and priority.

Key Features and Functionalities

  • Create Tasks: Users can add new tasks with details such as title, description, status, and priority.
  • Edit Tasks: Users can update existing tasks to reflect changes in details, status, or priority.
  • Filter Tasks: Users can filter tasks based on their status (e.g., pending, in-progress, completed) and priority (e.g., low, medium, high).
  • Delete Tasks: Users can remove tasks that are no longer needed.

Technologies Used

  • Frontend: Angular 17

Installation and Setup Instructions

Prerequisites

Node.js: Ensure you have Node.js v20.X.X installed. You can download it from the official Node.js website.

Steps

npm install
npm start

This will start the application and you can access it in your web browser at http://localhost:4200.

Usage Instructions

Creating a Task

Navigate to the "Create Task" section. Fill in the task details including title, description, status, and priority. Click the "Save" button to create the task.

Editing a Task

Select the task you want to edit from the task list. Update the necessary details. Click the "Update" button to save the changes.

Filtering Tasks

Use the filter options available to sort tasks based on their status and priority. The task list will update automatically to reflect the applied filters.

Deleting a Task

Select the task you want to delete. Click the "Delete" button to remove the task from the list.

Special Notes

Ensure your Node.js version is compatible with Angular 17 to avoid any compatibility issues. Regularly update your dependencies to the latest versions for optimal performance and security.
